# RestaurantDashboard

Система управления бронированиями столиков для ресторанов

## Описание
Desktop-приложение на JavaFX для управления бронированиями в ресторане. Автоматизация процесса бронирования, управление столиками в реальном времени, анализ загрузки зала.

## Функциональность
- Управление бронированиями (добавление, редактирование, удаление)
- Управление столиками и зонами ресторана
- Панель управления с сегодняшними бронированиями
- Автоматический поиск свободных столиков
- Статистика загрузки зала в реальном времени

## Технологии
- Java 8+
- JavaFX (GUI)
- SQLite (база данных)
- Maven (сборка)
