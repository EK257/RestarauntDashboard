# Обоснование выбора стека технологий

## Язык программирования: Java
- **Кроссплатформенность** — единая кодовая база для Windows, macOS, Linux
- **Статическая типизация** — повышение надежности и раннее выявление ошибок
- **Зрелая экосистема** — обширные библиотеки и инструменты для разработки
- **Управление памятью (JVM)** — снижение сложности разработки

## Фреймворк GUI: JavaFX
- **Современный API** — аппаратное ускорение, CSS-стилизация, FXML
- **Разделение логики и представления** — FXML для UI, Java для контроллеров
- **Scene Builder** — визуальное конструирование интерфейсов
- **Производительность** — подходит для динамического обновления данных

## База данных: SQLite
- **Встраиваемая архитектура** — не требует отдельного сервера
- **Простота развертывания** — единый файл базы данных
- **ACID-совместимость** — гарантии целостности данных о бронированиях
- **Минимальные требования** — эффективная работа на стандартном оборудовании

## Архитектурный паттерн: MVC
- **Четкое разделение ответственности** — модель, представление, контроллер
- **Упрощение тестирования** — изолированные компоненты
- **Масштабируемость** — возможность расширения функциональности

## Инструменты сборки: Maven
- **Стандартизация** — унифицированная структура проекта
- **Управление зависимостями** — автоматическое разрешение библиотек
- **Интеграция с IDE** — поддержка в IntelliJ IDEA, Eclipse, NetBeans

## Преимущества выбранного стека
1. **Минимальные требования к окружению** — только JRE 8+
2. **Автономная работа** — не требуется интернет-соединение
3. **Простота развертывания** — распределение в виде JAR-файла
4. **Надежность** — проверенные временем технологии
5. **Экономическая эффективность** — все компоненты с открытой лицензией
